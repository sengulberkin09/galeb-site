# Galeb — Ice Cream & Coffee 🍦

75 yıllık aile geleneğine sahip Galeb Dondurma (Bostanlı, Karşıyaka / İzmir) için
geliştirilmiş kurumsal web sitesi.

**🌐 Canlı site:** [galeb.com.tr](https://galeb.com.tr)

## Hakkında

1950'de Novi Pazar'da başlayan, bugün üçüncü kuşağın sürdürdüğü bir dondurma
geleneğinin dijital vitrini. Gerçek salep ve gerçek meyveyle üretilen el yapımı
dondurmanın hikâyesini, menüsünü ve aile arşivini bir araya getirir.
